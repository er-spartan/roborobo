# Forklift Project

This project uses both DC motors and Servo motors to be controlled using touch sensors (e.g., buttons) We program 8 touch sensors and assign action such as:

* Move left
* Move right
* Move forward
* Move backward
* Open claw
* Close claw
* Lift up arm
* Put down arm
